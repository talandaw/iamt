Color Artify

This function was fun and quick to do, based on the instructions given. It takes an image and replaces colors similar to each other with a block color, creating nifty stylized effect in images with lots of colors and pixel density. The formula was given, and this was a part of a group project, so it was fast to work on and complete.

                  def artify():
                      file = pickAFile()
                      pic = makePicture(file)
                      for x in range(0, getWidth(pic)):
                          for y in range(0 , getHeight(pic)):
                              p = getPixel(pic, x, y)

                              r = getRed(p)
                              if r < 64:
                                  setRed(p, 31)
                              elif r < 128:
                                  setRed(p, 95)
                              elif r < 192:
                                  setRed(p, 159)
                              else:
                                  setRed(p, 223)

                              g = getGreen(p)
                              if g < 64:
                                  setGreen(p, 31)
                              elif g < 128:
                                  setGreen(p, 95)
                              elif g < 192:
                                  setGreen(p, 159)
                              else:
                                  setGreen(p, 223)

                              b = getBlue(p)
                              if b < 64:
                                  setBlue(p, 31)
                              elif b < 128:
                                  setBlue(p, 95)
                              elif b < 192:
                                  setBlue(p, 159)
                              else:
                                  setBlue(p, 223)
                      show(pic)
                      return pic
