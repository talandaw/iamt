Red Eye Image Fix

This function was cool to work on but a pain to find useful pictures to test. It seemed that the values for red to our human eye can be so dramatically different to how the computer sees red, that I needed really dramatic examples. Luckily, I take way too many pictures of my dog and had some red eye ones on hand. The function below essentially checks pixel by pixel for colors close to red and then replaces them with a dark brown eye color.

After Picture:

Code:

                  def redEye():
                    file = pickAFile()
                    picture = makePicture(file)
                    eyeColorBrown = makeColor(43, 29, 14)
                    for px in getPixels(picture):
                      color = getColor(px)
                      if distance(color, red) < 100.0:
                        setColor(px, eyeColorBrown)
                    #writePictureTo(picture, folder+"redEyes.jpg")
                    show(picture)
                    return(picture)
