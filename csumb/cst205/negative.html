Make Negative

This function takes a picture and inverts it to make it a 'negative' of the original. It finds the original value and subtracts it from the full value (255) therefore leaving the opposite color. This function was very simple and easy to figure out.

Code:

                  def makeNegative():  
                    pic = get_pic()
                    pixels = getPixels(pic)
                    for p in pixels:
                      r = getRed(p)
                      g = getGreen(p)
                      b = getBlue(p)
                      setRed(p, 255 - r)
                      setGreen(p, 255 - r)
                      setBlue(p, 255 - r) 
                    repaint(pic)
                    show(pic)
