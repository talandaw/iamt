Shrink Image

This function takes an and shrinks it by half the height and width, resulting in a picture 1/4th of the original size. This one actually was complicated to troubleshoot because I happened to use an even height x width picture while developing it and then when I went to use it later as part of a larger program with an odd height I spent a while trying to figure out what my error was in the new code when it ended up being in this function! So I had to spend an hour or so revisiting what was wrong with this simple seeming function. Please note this website uses responsive images so depending on your viewpoint, the difference may not seem completely half.

Code:

                  def shrink(picture):
                  #edited to take into account dimensions both even and odd
                  #adding .5 and making an int to simulate math.ceil since we haven't learned  
                  #that yet but I believe that's what I would be using here otherwise
                    width = int(getWidth(picture)/2.0 +.5)
                    height = int(getHeight(picture)/2.0 +.5)
                    newPic = makeEmptyPicture(width, height)
                    for x in range (0, getWidth(picture), 2):
                      for y in range (0, getHeight(picture), 2):
                        setColor(getPixel(newPic, x/2, y/2), getColor(getPixel(picture, x, y)))
                    show(newPic)
                    return newPic
